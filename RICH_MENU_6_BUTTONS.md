# 🎨 Rich Menu Setup - 6 Buttons (Perfect Fit!)

## ✅ Use the 6-Grid Layout

You see template with **6 boxes (A, B, C, D, E, F)** - that's perfect!

---

## 📱 Setup in LINE Manager:

### 1. Go to: https://manager.line.biz/
### 2. Select your account (2010529677)
### 3. Chat → Rich menus → Create

---

## 🎯 Configure 6 Buttons:

### Template: 
Choose **6 boxes template** (2 rows × 3 columns)

```
┌─────────┬─────────┬─────────┐
│    A    │    B    │    C    │
│ Recent  │ Summary │Categories│
├─────────┼─────────┼─────────┤
│    D    │    E    │    F    │
│Log Cash │ Export  │  Help   │
└─────────┴─────────┴─────────┘
```

---

## ⚙️ Configure Each Area:

Click each box (A, B, C, D, E, F) and set:

| Box | Action Type | Text to Send |
|-----|-------------|--------------|
| **A** | ข้อความ (Message) | `recent` |
| **B** | ข้อความ (Message) | `summary` |
| **C** | ข้อความ (Message) | `categories` |
| **D** | ข้อความ (Message) | `log cash` |
| **E** | ข้อความ (Message) | `export` |
| **F** | ข้อความ (Message) | `help` |

---

## 🎨 Background:

**Option 1: Solid Color (Easiest)**
- Select "สีเดียว" (Solid color)
- Choose any color: Blue, Green, Purple
- The menu bar text "เมนู" will show

**Option 2: Simple Image**
- Create 2500 x 1686 px image
- Add text labels for each button
- Save as PNG/JPG

---

## ✅ Activate:

1. Click **บันทึก** (Save)
2. Make sure toggle is **เปิด** (ON)
3. Set as **เริ่มต้น** (Default)

---

## 🧪 Test:

Open LINE app → Bottom menu appears with 6 buttons! 🎉

**Tap "Summary"** → Bot shows monthly breakdown
**Tap "Recent"** → Shows last 5 transactions
**Tap "Log Cash"** → Instructions for cash logging

---

## 💡 The "Help" button is bonus!

When tapped, bot shows all available commands:
```

```
